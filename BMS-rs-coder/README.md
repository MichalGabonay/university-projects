Dva konzolové programy (bms2A - kodér, bms2B - dekodér) v jazyku C++, ktoré implementujú vhodné zabezpečovacie kódy (Error Correction Code). Pre samotné ECC využívam Reed-Solomon s prekladaním bitov. Dosiahnutá 16-18% error correction u random aj burst mode.
